Import module:
    >>> MyList = __import__('1-my_list').MyList

Without elements:
    >>> m = MyList()
    >>> print(m)
    []

One int element:
    >>> m = MyList()
    >>> m.append(1)
    >>> print(m)
    [1]

More than one element:
    >>> m = MyList()
    >>> m.append(1)
    >>> m.append(2)
    >>> m.append(3)
    >>> print(m)
    [1, 2, 3]

Try print_sorted with sorted apppend:
    >>> m = MyList()
    >>> m.append(1)
    >>> m.append(2)
    >>> m.append(3)
    >>> m.print_sorted()
    [1, 2, 3]

Try print_sorted with not sorted apppend:
    >>> m = MyList()
    >>> m.append(5)
    >>> m.append(3)
    >>> m.append(6)
    >>> m.print_sorted()
    [3, 5, 6]

Try print_sorted with not sorted apppend content negative numbers:
    >>> m = MyList()
    >>> m.append(1)
    >>> m.append(-1)
    >>> m.append(0)
    >>> m.print_sorted()
    [-1, 0, 1]

Try print_sorted with sorted apppend all numbers are negative:
    >>> m = MyList()
    >>> m.append(-1)
    >>> m.append(-2)
    >>> m.append(-3)
    >>> m.print_sorted()
    [-3, -2, -1]

Try print_sorted with sorted apppend content negative numbers:
    >>> m = MyList()
    >>> m.append(-1)
    >>> m.append(0)
    >>> m.append(1)
    >>> m.print_sorted()
    [-1, 0, 1]

Try print_sorted with empty list:
    >>> m = MyList()
    >>> m.print_sorted()
    []
